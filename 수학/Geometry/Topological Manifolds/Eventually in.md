# Eventually in
Topological space $X$와 $X$의 subset $A$가 있다고 하자.

$X$위의 sequence $s$가 다음을 만족할 떄, $s$를 `eventually in` $A$라고 한다.
$$ \text{for all } i \text{ but finitely many values of } i, \quad s(i) \in A$$
