Strong formulation

경계값 문제(boudnary value problem)가 다음과 같이 주어졌다고 하자.

find u∈Us.t.P(u)+f(x)=0in Ω⊂Rd\begin{equation} \text{find } u \in \mathcal U \quad s.t. \quad \mathcal P(u) + f(\mathbf x) = 0 \quad \text{in } \Omega \subset \R^d \end{equation} find u∈Us.t.P(u)+f(x)=0in Ω⊂Rd​​

Where, U:={u∈Cm(Ω)∣u satisfies boundary condition on ∂Ω}\text{Where, } \mathcal U := \{ u \in C^m(\Omega) \enspace | \enspace u \text{ satisfies boundary condition on } \partial\Omega \} Where, U:={u∈Cm(Ω)∣u satisfies boundary condition on ∂Ω}

이 때, P:Cm(Ω)→C0(Ω)\mathcal P : C^m(\Omega) \rightarrow C^0(\Omega)P:Cm(Ω)→C0(Ω)는 계수(order)가 mmm인 미분 연산자(differential operator)이며 U\mathcal UU는 solution funtion space이다.

경계조건(boundary condition; BC)은 ∂ΩE⊂∂Ω\partial\Omega_E \subset \partial\Omega∂ΩE​⊂∂Ω따라 essential BC와 ∂ΩN⊂∂Ω\partial\Omega_N \subset \partial\Omega∂ΩN​⊂∂Ω따라 natural BC가 주어져있으며 ∂Ω=∂ΩE∪∂ΩN\partial\Omega = \partial\Omega_E \cup \partial\Omega_N∂Ω=∂ΩE​∪∂ΩN​ 이다.

식(1)을 PDE의 strong formulation이라 하는데 이는 strong formulation의 해가 Ω\OmegaΩ의 모든 점에서 PDE를 만족시켜야하며 mmm번 미분 가능해야된다는 엄격한 정규성(regularity)을 만족시켜야하기 때문이다.

Weighted residual formulation

식(1)의 weighted residual formulation은 다음과 같이 주어진다.

find u∈Us.t.∀w∈W,∫Ωwr dV=0\begin{equation} \text{find } u \in \mathcal U \quad s.t. \quad \forall w \in \mathcal W, \quad \int_\Omega w r \thinspace dV = 0 \end{equation} find u∈Us.t.∀w∈W,∫Ω​wrdV=0​​

Where, Cc∞(Ω)⊆W,r(x)=P(u)+f(x)\text{Where, } C^\infty_c(\Omega) \subseteq \mathcal W, \quad r(\mathbf x) = \mathcal P(u) + f(\mathbf x) Where, Cc∞​(Ω)⊆W,r(x)=P(u)+f(x)

이 떄, W\mathcal WW는 테스트 함수공간(test function space), www는 테스트 함수(test function), rrr은 residual이라한다.

이를 weighted residual formulation이라고 부르는 이유는 식(2)가 residual의 가중평균이 0이 되는 것과 같은 형태를 가지고 있기 때문이다.

그렇다고 식(2)의 해가 r=0r = 0r=0을 만족하는게 아니라 가중평균만 000으로 보낸다고 오해하면 안된다. 식(2)는 식(1)과 동치이며 이는 변분법(variation calculus)의 기본 보조정리에 의해 증명된다.

일반적으로 weighted residual formulation은 functional Br,lrB_r,l_rBr​,lr​을 이용해 다음과 같이 간단하게 나타낼 수 있다.

find u∈Us.t.∀w∈W,Br(w,u)=lr(w)\begin{equation} \text{find } u \in \mathcal U \quad s.t. \quad \forall w \in \mathcal W, \quad B_r(w,u) = l_r(w) \end{equation} find u∈Us.t.∀w∈W,Br​(w,u)=lr​(w)​​

Where, Br(w,u):=∫ΩwP(u) dV,lr(w):=−∫Ωwf dV\text{Where, } B_r(w,u) := \int_\Omega w\mathcal P(u) \thinspace dV, \quad l_r(w) := -\int_\Omega wf \thinspace dV Where, Br​(w,u):=∫Ω​wP(u)dV,lr​(w):=−∫Ω​wfdV

이 때, BrB_rBr​은 첫번째 원소에 대해서 linear map이며, lrl_rlr​은 linear map이다.

만약, P\mathcal PP가 linear operator라면, BrB_rBr​은 bilinear map이 된다.
