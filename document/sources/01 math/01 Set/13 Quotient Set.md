# Quotient Set
## Definition
Set $S$와 equivalence relation $\sim \subseteq S \times S$이 있다고 하자.

`상집합(quotient set)` $S/\sim$은 다음과 같이 정의되는 집합이다.

$$ S/\sim := \Set{ [s] | s \in S } $$

즉, equivalence class의 family이다.

### 참고
Quotient set은 정의상 equivalence relation을 요구한다.
